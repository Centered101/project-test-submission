เอกสารของ Project-Test-Submission

ภาพรวม

Project-Test-Submission เป็นแพลตฟอร์มที่จัดทำขึ้นเพื่อรวบรวมโปรเจกต์ที่ผู้พัฒนาได้สร้างสรรค์หรือศึกษาวิจัยไว้ โดยมีรูปแบบเป็น ไดเรกทอรีรายการ (Directory Listing) ช่วยให้ผู้เข้าชมสามารถค้นหาและเข้าถึงข้อมูลโค้ดของแต่ละโปรเจกต์ได้อย่างสะดวกและเป็นระบบ

คุณสมบัติ

  • แสดงโปรเจกต์ในรูปแบบรายการ: แสดงโปรเจกต์ทั้งหมดพร้อมรายละเอียดและลิงก์สำหรับเข้าชม
  • ระบบค้นหาและกรองข้อมูล: รองรับการค้นหาโปรเจกต์เพื่อให้เข้าถึงสิ่งที่ต้องการได้รวดเร็ว
  • รองรับการใช้งานทุกอุปกรณ์: ออกแบบด้วย Tailwind CSS เพื่อให้แสดงผลได้อย่างเหมาะสมทั้งบนมือถือ แท็บเล็ต และคอมพิวเตอร์
  • ธีมหลัก: ใช้โทนสีน้ำเงิน-ขาว #409EFE และ #FFFFFF
  • โครงสร้าง URL ที่เหมาะสมกับ SEO: ช่วยเพิ่มประสิทธิภาพในการค้นหาผ่านเครื่องมือค้นหา

เทคโนโลยีที่ใช้

  • ส่วนหน้า (Frontend): HTML, CSS, JavaScript, TailwindCSS, JQuery
  • ส่วนหลัง (Backend): ไม่มี
  • ฐานข้อมูล (Database): ไม่มี
  • โฮสติ้ง (Hosting): เดิมใช้ Firebase Hosting ปัจจุบันโฮสต์บน Netlify

วิธีการใช้งาน

  • เข้าเว็บไซต์หลักของ Project-Test-Submission
  • เลือกโปรเจกต์ที่สนใจเพื่อศึกษา
  • อ่านรายละเอียดของโปรเจกต์และดูตัวอย่างโค้ด
  • หากต้องการทดลองรันโค้ด สามารถใช้แพลตฟอร์มภายนอก เช่น Replit หรือ GitHub Codespac

แผนการพัฒนาในอนาคต

  • เพิ่มฟังก์ชันให้ผู้ใช้สามารถเพิ่มโปรเจกต์ของตนเองได้
  • รองรับระบบสมัครสมาชิกและเข้าสู่ระบบ
  • จัดทำ API สำหรับดึงข้อมูลโปรเจกต์
  • พัฒนาเวอร์ชันใหม่ด้วย React + Vite
  • พัฒนาต่อยอดเป็น Full Stack ด้วย MERN Stack
