About GitHub Profile API

This tool allows you to search and explore GitHub user profiles with detailed information including repositories, followers, following, and contribution statistics.

Built with GitHub's REST API to provide real-time data about any public GitHub user profile.

Features:

  • • Search any GitHub user
  • • View profile statistics
  • • Repository information
  • • Follower/Following counts
  • • Account creation date
  • • Location and bio information
