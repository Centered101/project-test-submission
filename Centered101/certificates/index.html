นาย พงศ์พล พรมผา

สวัสดีครับ ผมชื่อเซ็นเตอร์ ปัจจุบันกำลังศึกษาอยู่ในระดับชั้น ปวช.2/2 สาขาเทคโนโลยีธุรกิจดิจิทัล ที่วิทยาลัยเทคโนโลยีเอเชียลพบุรี

ผมมีความสนใจด้านการพัฒนาเว็บไซต์และแอปพลิเคชัน โดยเน้นการใช้เทคโนโลยีสมัยใหม่ เช่น JavaScript, React และ Tailwind CSS

นอกจากด้านเทคโนโลยีแล้ว ผมยังหลงใหลในเรื่องของการลงทุน หุ้น และอสังหาริมทรัพย์ โดยมีเป้าหมายในการสร้างอิสรภาพทางการเงินและเป็นเจ้าของธุรกิจของตัวเองในอนาคต
